Image Captioning using Deep Learning

Image captioning aims to generate a descriptive sentence for an image using AI models...

Image captioning aims to generate a sentence description for an image automatically. Using a Convolutional Neural Network (CNN) as an encoder and a Long Short Term Memory (LSTM) model as a decoder, the project produces English sentences that describe the contents of an image. The model's effectiveness is evaluated using BLEU scores, which assess the accuracy of generated captions against human-written references.
